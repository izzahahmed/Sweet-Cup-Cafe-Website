# ☕ Sweet Cup Cafe Website

Welcome to my first front-end development project!

Sweet Cup Cafe is a responsive multi-page cafe website designed using HTML, CSS, and Bootstrap. It features a soft pink-and-white theme and provides users with an engaging browsing experience.

## 🌸 Features

- Responsive design using Bootstrap
- Home page
- Menu page
- Reservation page
- Contact page
- Navigation bar
- HTML forms for table reservations
- Custom CSS styling
- Clean and beginner-friendly design

## 💻 Technologies Used

- HTML5
- CSS3
- Bootstrap 5

## 🎯 Purpose

This project was created to practice front-end web development concepts learned during my studies and to improve my skills in building responsive websites.

Thank you for visiting my project! 🌸
